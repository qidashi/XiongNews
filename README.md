# 仿今日头条
《爱上Android开发》学习笔记。

技术栈：
> * 主界面：tabLayout + viewPager
> * 秘钥获取：NDK
> * 网络请求：retrofit2 + rxandroid
> * 数据解析：gson
> * 新闻列表：ListView + 自定义SwipeRefreshLayout
> * 图片请求及展示：picasso
